async handle_data_message(message)¶

Handle a data message.

Sends the message to the corresponding groups based on the data of the message.

message: dict

dictionary containing the message parsed as json.

The expected format of the message for a telemetry or an event is as follows: {

category: ‘event’/’telemetry’, data: [{

csc: ‘ScriptQueue’, salindex: 1, data: {

stream1: {….}, stream2: {….},

}

}]

}

The expected format of the message for a command is as follows: {

category: ‘cmd’, data: [{

csc: ‘ScriptQueue’, salindex: 1, data: {

cmd: ‘CommandPath’, params: {

‘param1’: ‘value1’, ‘param2’: ‘value2’, …

},

}

}]

}

async handle_subscription_message(message)¶

Handle a subscription/unsubscription message.

Makes the consumer join or leave a group based on the data from the message

message: dict

dictionary containing the message parsed as json.

The expected format of the message is as follows: {

option: ‘subscribe’/’unsubscribe’ category: ‘event’/’telemetry’/’cmd’, csc: ‘ScriptQueue’, salindex: 1, stream: ‘stream1’,

}

async receive_json(message)¶

Handle a received message.

Calls handle_subscription_message() if the message is intended to join or leave a group. Otherwise handle_data_message() is called

message: dict

dictionary containing the message parsed as json
